what is java script ?

JavaScript is a programming language that is used to add dynamic functionality and complex features like interactivity , It is a scripting language that enables you to create dynamically updating content, control multimedia, animate images, and pretty much everything else.

In simpler terms, JavaScript is a language that allows developers to create interactive web pages that respond to user actions, without requiring the page to be reloaded from the server. It is a client-side language, meaning that it runs on the user’s computer rather than on the server.

An undefined variable is one that has been declared, but has not been assigned any value. The default value of an unassigned variable

what is === operator ?

The === operator in JavaScript is called the strict equality operator. It checks whether its two operands are equal in both value and type.

For example, 9 === 9 is true, but "9" === 9 is false, because the string "9" is not the same type as the number 9.

what are all the looping structure in java script ?

Looping structures are ways of repeating a block of code based on a condition. JavaScript supports different kinds of loops, such as:

for loop: This loop executes a block of code a fixed number of times, or until a specified condition is false. It has three expressions: initialization, testing, and increment/decrement. For example: 

      // This loop prints the numbers from 1 to 10
    
      for (let i = 1; i <= 10; i++) { 
        console.log(i); 
      }

while loop: This loop executes a block of code as long as a specified condition is true. It has only one expression: testing. For example: 

          // This loop prints the numbers from 1 to 10 

          let i=1; while (i<=10)
          { 
            console.log(i); i++; 
          }

do…while loop: This loop is similar to the while loop, but it executes the block of code at least once before checking the condition. It has two expressions: testing and body. For example: 

          // This loop prints the numbers from 1 to 10 

          let i=1; 
          
          do { 
            console.log(i); i++; 
          } while (i <=10);

what are all types of pop up boxes available in java script ?

There are three types of pop-up boxes in JavaScript: alert, confirm, and prompt.

Alert box: It shows a message and an OK button. The user has to click OK to proceed.

alert ("message");


Confirm box: It shows a message and OK and Cancel buttons. The user can choose either option to proceed. The box returns true if OK is clicked, and false if Cancel is clicked.

confirm ("message");


Prompt box: It shows a message and a text input field. The user can enter a value and click OK or Cancel to proceed. The box returns the entered value if OK is clicked, and null if Cancel is clicked.

prompt("message", "default value");
