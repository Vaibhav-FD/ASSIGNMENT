what is JQUERY ?

jQuery is an open source JavaScript library that simplifies the interactions between an HTML/CSS document

The purpose of jQuery is to make it much easier to use JavaScript on your website.

jQuery is a lightweight, "write less, do more", JavaScript library.

how to apply css using JQuery , how to add class and remove class in JQuery , JQuery Animation ?

To apply CSS properties to an element using jQuery, you can use the css() method. For example, you can change the color of a paragraph to pink by using:

$("p").css("color", "pink"); $("p").css(color : "pink");


To add or remove a CSS class to an element using jQuery, you can use the addClass() and removeClass() methods. These methods can add or remove one or more classes from the selected elements. For example, you can add a class named “test” to a div by using:

$("div").addClass("test");


To create animations using jQuery, you can use the animate() method. This method can animate the CSS properties of the selected elements over a specified duration. For example, you can animate the width of a div to 912 pixels by using:

$("div").animate({width: "912px"});

How to create slider with animation ?

The “silder” is a basic jQuery plugin that allows you to create animated image slider. You just need to place your images in slider’s HTML structure. After that, plugin render a beautiful fullscreen animated image slider with next previous buttons. Moreover, it can be also customized with additional CSS according to your needs.
